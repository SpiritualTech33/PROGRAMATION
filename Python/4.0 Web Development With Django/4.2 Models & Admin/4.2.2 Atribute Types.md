Well, as i said before, the atribute types are this declarative forms that indicates what type of data each column receives. It's like you're telling the program: This column just receives integer numbers.



## 📝 Text & Web Fields

These fields handle strings and provide built-in validation for specific formats.

|**Field Type**|**Purpose**|**Key Detail**|
|---|---|---|
|`CharField`|Short text (titles, names)|**Required**: `max_length`|
|`TextField`|Long text (descriptions, content)|No practical character limit|
|`SlugField`|URL-friendly strings|Implies an index for faster lookups ⚡|
|`EmailField`|Email addresses|Automatically validates the "@" format|
|`URLField`|Web addresses|Validates for `http://` or `https://`|

## 🔢 Numeric Fields

Choosing the right numeric field ensures data integrity and saves storage space.

|**Field Type**|**Purpose**|**Best Practice**|
|---|---|---|
|`IntegerField`|Whole numbers|Standard for most counts|
|`PositiveIntegerField`|Numbers ≥ 0|Use for age, stock, or page counts|
|`DecimalField`|Exact decimals|**Use for Money** 💰 (Requires `max_digits` and `decimal_places`)|
|`FloatField`|Scientific decimals|Avoid for financial data due to rounding errors|
|`AutoField`|ID numbers|Increments automatically (Django adds this by default)|

## 📅 Date & Time Fields

|**Field Type**|**Python Data Type**|**Stores...**|
|---|---|---|
|`DateField`|`datetime.date`|Just the date (Year, Month, Day)|
|`TimeField`|`datetime.time`|Just the time (Hours, Minutes, Seconds)|
|`DateTimeField`|`datetime.datetime`|Both date and time|

Django can manage timestamps automatically using these two parameters:

> [!TIP] Automation Parameters
> 
> - **`auto_now_add=True`**: Set once when the object is created. Great for "Created At".
>     
> - **`auto_now=True`**: Updates every time the object is saved. Great for "Last Modified".
>

## 🔗 Relationship Fields

These fields define how different tables connect to one another.

- **`ForeignKey(Model, on_delete=...)`**: Many-to-One (e.g., Many Posts belong to one Author).

- **`ManyToManyField(Model)`**: Many-to-Many (e.g., A Book has many Tags, and a Tag has many Books).

- **`OneToOneField(Model, on_delete=...)`**: One-to-One (e.g., One User has one Profile).


---

## 📂 File & Image Fields

- **`FileField(upload_to='path/')`**: Stores a link to a file in your storage.

- **`ImageField(upload_to='path/')`**: Same as FileField but validates that the upload is a valid image (requires the `Pillow` library).




After seeing this, you might feel a little bit overwhelmed,  but don't worry, you don't have to learn all this now. You gonna learn how to use it in the praxis. By now keep with this basic ones.

|Field Type|What it stores|When to use it|Common parameters|
|---|---|---|---|
|**CharField**|Short text (max 255 chars usually)|Names, titles, usernames|`max_length` (required)|
|**TextField**|Long text (unlimited)|Comments, descriptions, blog posts|`blank=True`|
|**IntegerField**|Whole numbers|Age, count, rating (1-5)|`default`, `validators`|
|**FloatField**|Decimal numbers|Price, temperature, score|`default`|
|**BooleanField**|True or False|Yes/No flags (e.g. is_published)|`default=False`|
|**DateTimeField**|Date + time|Created at, last login|`auto_now_add=True`, `default`|
|**DateField**|Only date|Birth date, event day|`auto_now_add=True`|
|**EmailField**|Email address|User email|`max_length=254`|
|**URLField**|Website link|External links|`max_length=200`|
|**ForeignKey**|Link to another model (one-to-many)|"This review belongs to this book"|`on_delete`, `related_name`|

```python
# myapp/models.py
from django.db import models
from django.utils import timezone

class Book(models.Model):
    """A simple book in our library app."""
    
    title = models.CharField(
	    max_length = 200
	    help_text = "Short text - book title"
    )
    
    author = models.CharField(
	    max_length = 100
	    help_text = "Short text - Book author"
    )
    
    published_year = models.IntegerField(
	    help_text = "Whole number - Year only"
    )
    
    price = models.FloatField(
	    null = True, blank = True,
	    help_text = "Decimal number - Can be empty"
    )
    
    is_available = models.BooleanField(
	    default = True
	    help_text = "True/False flag"
    )
    
    created_at = models.DateTimeField(
	    auto_now_add = True,
	    help_text = "Automatically set when book is added"
    )
    
    def __str__(self): 
	    return f"{self.title} by {self.author}"
```
