What's the difference between an array and a list?
Okey, let's make this simple and straight. A list is a collection in python, usually used as a data structure, is simple in his syntax, and very flexible, which means that you can save any data type that you want inside. Integers, float, strings and Boolean values. In the other hand, the arrays only accept one data type. In the context of NumPy, this data type are the numbers.

### **COMPARISON TABLE:**

| Feature                     | Python List         | NumPy Array                |
| --------------------------- | ------------------- | -------------------------- |
| **Type flexibility**        | Mixed types ✅       | Single type only           |
| **Speed (math ops)**        | Slow 🐌             | Fast ⚡ (10-100x)           |
| **Memory efficiency**       | Heavy 📦            | Compact 📊                 |
| **Vectorization**           | No                  | Yes ✅                      |
| **Mathematical operations** | Manual loops        | Built-in                   |
| **Use case**                | General programming | Numerical computation      |
| **Resizing**                | Easy (append, pop)  | Costly (creates new array) |
