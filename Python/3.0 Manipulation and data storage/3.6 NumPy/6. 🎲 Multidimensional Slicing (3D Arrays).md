Now, we're heading to the majors, slicing 3D arrays. 

Think of it like:
- **1D** = A line of numbers
- **2D** = A table (rows × columns)
- **3D** = Multiple tables stacked (depth × rows × columns)

Usually, this is called a Tensor.

# CREATING A 3D ARRAY

```python
import numpy as np 

# 3D ARRAY --> 3 MATRICES STACKED
tensor = np.array([
	#Matrix 0 (depth 0)
	[[1, 2, 3],
	[6, 9, 12]],
	
	# Matrix 1 (depth 1)
	[[25, 12, 11],
	[5, 6, 7]],
	
	# Matrix 2 (depth 2)
	[[9, 12, 15],
	[18, 24, 30]]
])

tensor_shape = tensor.shape
print(tensor_shape) # (3, 2, 3)
```
**Shape = (depth, rows, columns)**

- 3 matrices (depth)
- 2 rows per matrix
- 3 columns per matrix


# 3D INDEXING SYNTAX 

```python
tensor[depth, row, column]
```




# ACCES THE ENTIRE MATRIX (ONE LAYER)

```python
# 3D ARRAY --> 3 MATRICES STACKED
tensor = np.array([
	#Matrix 0 (depth 0)
	[[1, 2, 3],
	[6, 9, 12]],
	
	# Matrix 1 (depth 1)
	[[25, 12, 11],
	[5, 6, 7]],
	
	# Matrix 2 (depth 2)
	[[9, 12, 15],
	[18, 24, 30]]
])

# GET FIRST MATRIX
first_matrix = tensor[0]

# GET SECOND MATRIX
second_matrix = tensor[1]

# EXPLICIT WAY (same result)
#GET THIRD MATRIX
third_matrix = tensor[2, :, :]


```


# ACCES SPECIFIC ELEMENT

```python

# 3D ARRAY --> 3 MATRICES STACKED
tensor = np.array([
	#Matrix 0 (depth 0)
	[[1, 2, 3],
	[6, 9, 12]],
	
	# Matrix 1 (depth 1)
	[[25, 12, 11],
	[5, 6, 7]],
	
	# Matrix 2 (depth 2)
	[[9, 12, 15],
	[18, 24, 30]]
])

# ELEMENT AT DEPTH 2, row 0, column 2.
choosen_element = tensor[2:0:2]
print(choosen_element) # 15

# ELEMENT AT DEPTH 0, row 1, column 0
choosen_element = tensor[0:1:0]
print(choosen_element) # 6
```



# SLICE ACROSS DEPTH
```python

```python
tensor = np.array([
	#Matrix 0 (depth 0)
	[[1, 2, 3],
	[6, 9, 12]],
	
	# Matrix 1 (depth 1)
	[[25, 12, 11],
	[5, 6, 7]],
	
	# Matrix 2 (depth 2)
	[[9, 12, 15],
	[18, 24, 30]]
])

# GET FIRST TWO MATRICES (depth 0 and 1)
two_matrices = tensor[0:2]

# OR EXPLICIT
two_matrices = tensor[0:2, :, :]
```


# GET SAME POSITION ACROSS ALL DEPTHS

```python
# Element [0, 0] from ALL matrices
element_00_all = tensor[:, 0, 0]
print(element_00_all)  # [ 1  7 13]

# First row from ALL matrices
first_row_all = tensor[:, 0, :]
print(first_row_all)
# [[ 1  2  3]
#  [ 7  8  9]
#  [13 14 15]]

# First column from ALL matrices
first_col_all = tensor[:, :, 0]
print(first_col_all)
# [[ 1  4]
#  [ 7 10]
#  [13 16]]
```

## **Visual Mental Model:**
```
tensor[depth, row, column]

:           = "all"
0:2         = "first 2"
1           = "only index 1"

Examples:
tensor[0, :, :]     # Matrix 0 (all rows, all cols)
tensor[:, 0, :]     # First row from each matrix
tensor[:, :, 0]     # First column from each matrix
tensor[1, 1, 2]     # Specific single element
```



## **Quick Reference:**

```python
# 3D array
arr = np.array(shape=(3, 4, 5))  # depth=3, rows=4, cols=5

arr[0]           # First matrix (all rows, all cols)
arr[:, 0, :]     # First row from all matrices
arr[:, :, 0]     # First column from all matrices
arr[1, 2, 3]     # Single element
arr[0:2, :, :]   # First 2 matrices
```
