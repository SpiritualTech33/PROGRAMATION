Its a way to select values, using conditions to filter them. 
Like saying: "Give me only players with batting average > .300"

# HOW IT WORKS?

```python
import numpy as np

batting_averages = np.array([0.304, 0.287, 0.311, 0.295, 0.278])

elite_hitters = batting_averages[batting_averages > 0.290]
print(elite_hitters)
```


# DIFFERENT CONDITIONS

```python
home_runs = np.array([46, 62, 40, 35, 32, 28, 21])

# Greater than
sluggers = home_runs[home_rus > 40]
print(sluggers)

# Less than
contact_hitters = home_runs[home_runs < 15]
print(contact_hitters)

# Equal to
american_league_record = home_runs[home_runs == 62]
print(american_league_record)

# Not equal to
not_40 = home_runs[home_runs != 40]

# Greater than or equal
at_least_40 = home_runs[home_runs >= 40]

```



# Multiple Conditions - AND(&)

```python
batting_avg = np.array([0.304, 0.287, 0.311, 0.295, 0.278]) 
home_runs = np.array([46, 62, 32, 35, 21])

# BOTH conditions must be True # BA > .290 AND HR > 30
elite_sluggers = batting_avg[(batting_avg > 0.295) & (home_runs > 35)]
print(elite_sluggers)
```
CRITICAL: Use parentheses around EACH condition!


# Multiple Conditions - OR (`|`)

```python
# At least ONE condition must be True
# BA > .300 OR HR > 40
good_players = batting_avg[(batting_avg > 0.300) | (home_runs > 40)]

print(good_players)  # [0.304 0.287 0.311]
```


# NOT (`~`) - Negation

```python
# NOT struggling (BA >= .250)
not_struggling = batting_avg[~(batting_avg < 0.250)]

print(not_struggling)  # All of them (none below .250)

# NOT power hitters (HR <= 35)
not_power = home_runs[~(home_runs > 35)]

print(not_power)  # [32 35 21]
```

## **Operators Summary:**

python

```python
&    # AND (both conditions True)
|    # OR (at least one True)
~    # NOT (flip True/False)

# ALWAYS use parentheses!
arr[(arr > 5) & (arr < 10)]   # ✅ Correct
arr[arr > 5 & arr < 10]        # ❌ Error!
```
