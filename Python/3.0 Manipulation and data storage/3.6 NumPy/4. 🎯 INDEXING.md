
# INDEXING
## **What IS Indexing?**

**Pointing at ONE specific element in an array.**

Like saying: "Give me the 3rd player's batting average."


 
 # 1D ARRAY INDEXING

```python
import numpy as np

batting_averages = np.array([0.304, 0.287, 0.311, 0.295, 0.278])

# ACCES THE FIRST ELEMENT (index 0)
first_value = batting_averages[0]
print(first_value) # 0.304

# ACCES THE THIRD ELEMENT
third_value = batting_averages[2]
print(third_value) # 0.311

# ACCES TO THE LAST ELEMENT
last_value = battinng_averages[-1]
print(last_value) # 0.278

# ACCES SECOND TO LAST ELEMENT
second_last = batting_averages[-2]
print(second_last) # 0.295

```
**Rule:** Arrays start at index `0`, negative indices count from end.



# 2D ARRAY INDEXING 

```python
# Matrix: 3 players, 3 stats (BA, HR, RBI) 
stats = np.array([ 
	[0.304, 46, 100], # Player 0 
	[0.287, 62, 131], # Player 1 
	[0.311, 32, 97] # Player 2 
])

# Syntax: array[row, column]

# GET ELEMENT AT ROW 0 AND COLUMN 0
player_zero_ba = stats[0,0]
print(player_zero_ba) # 0.304

# GET ELEMENT AT ROW 1 COLUMN 1
player_one_homeruns = stats[1,1]
print(player_one_homeruns) # 62

# GET ELEMENT AT ROW 0 COLUMN 2
player_zero_rbis = stats[0,2]
print(player_zero_rbis) # 100
```
**Syntax:** `array[row, column]` (both start at 0)


# GET ENTIRE ROW

```python
# GET ALL STATS FROM PLAYER 1
player_one_stats = stats[1, :]
# The `:` means "all columns"
```


# GET THE ENTIRE COLUMN
```python
# Get all batting averages (column 0) 
all_batting_avgs = stats[:, 0]
 print(all_batting_avgs) # [0.304 0.287 0.311] 
 
 # Get all home runs (column 1) 
 all_home_runs = stats[:, 1] 
 print(all_home_runs) # [46. 62. 32.]
 
 # The `:` means "all rows"
```
