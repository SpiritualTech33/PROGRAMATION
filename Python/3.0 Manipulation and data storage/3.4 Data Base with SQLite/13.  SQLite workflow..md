The Complete SQLite Workflow in Python

### Step 1: Import the Library

```python
import sqlite3
```
Import the library that allow us to work with SQLite, it's integrated with python. No installation needed.



### Step 2: Connect to Database
```python
# Creates database file if it doesn't exist, opens if it does
connection = sqlite3.coonect('philosophy.db')
```
**What happens:**
- If `philosophy.db` exists → opens it
- If it doesn't exist → creates it
- Returns a **connection object** (your bridge to the database)

Special case - Memory database:
```python
connection = sqlite3.connect(':memory:')
# Creates temporary database in RAM (disappears when program ends)
# Useful for testing
```



### Step 3: Create a Cursor
```python
cursor = connection.cursor()
```
**What is a cursor?**
Think of it as your **pen** for writing to the database. The connection is the **notebook**, the cursor is the **tool you use to write**.

**Cursor = execution interface**



### Step 4: Enable Foreign Keys (SQLite Specific!)
```python
cursor.execute("PRAGMA foreign_keys = ON;")
```
**Critical:** SQLite doesn't enforce foreign keys by default. Always enable them if you're using relationships.



### Step 5: Create Tables
```python
cursor.execute("""
	CREATE TABLE IF NOT EXISTS philosophers (
		philosopher_id INTEGER PRIMARY KEY AUTOINCREMENT,
		name TEXT NOT NULL
		birth_year INTEGER
		death_year INTEGER
		school_of_thought TEXT
	)
""")
```
**Note:** `IF NOT EXISTS` prevents errors if table already exists.



### Step 6: Insert Data - Parameterized (SAFER - prevents SQL injection)
```python
philosopher = ('Kant', 1724, 1804, 'Trascendental Idealism')

cursor.execute ("""
	INSERT INTO philosophers (name, birth_year, death_year, school_of_thought)
	VALUES (?,?,?,?)
""", philosopher)
```

Insert Many
```python
philosophers = [ 

('Socrates', -470, -399, 'Classical'),

	VALUES (?, ?, ?, ?)

```
**The `?` placeholders:** SQLite safely inserts your values, preventing SQL injection attacks.


### Step 7: Query Data (SELECT)
```python
cursor.execute("SELECT * FROM philosophers") 
results = cursor.fetchall() 

for row in results: 
	print(row)
```
OUTPUT
(1, 'Socrates', -470, -399, 'Classical'), 
(2, 'Descartes', 1596, 1650, 'Rationalism'), 
(3, 'Nietzsche', 1844, 1900, 'Existentialism')


FETCH WITH FILTERING
```python
cursor.execute(""" 
	SELECT name, school_of_thought 
	FROM philosophers WHERE birth_year < 0 
""") 
ancient_philosophers = cursor.fetchall() 

for name, school in ancient_philosophers: 
	print(f"{name}: {school}")
```



### Step 8: Close Connection
 **The BETTER Way: Using `with` Statement**
```python
import sqlite3 

with sqlite3.connect('philosophy.db') as connection: 
	cursor = connection.cursor()
```
This prevents any error and closes automatically when the code block is done.

## **Key Principles:**

1. **Always use parameterized queries** (`?` placeholders) - prevents SQL injection
2. **Always commit** to make changes permanent
3. **Use `with` statement** for automatic cleanup
4. **Enable foreign keys** if using relationships
5. **Use `IF NOT EXISTS`** to prevent errors
