JOIN is a command that allow us to connect 2 tables in one single result. Imagine that you take 2 separate datos from each table and Join them into a single temporary result.
This is like this, because, usually we have different tables with different info that needs to be match. 
Is a SQL that reconstructs the whole from fragmented parts using the foreign keys.

The most common usage is with INNER JOIN, returns rows where the relationship exists in both tables. And LEFT JOIN that matches relation even if is missing data.

```python
import sqlite3 

with sqlite3.connect('empresa.db') as conn: 
	cursor = conn.cursor() 
	
	print("="*60) 
	print("INNER JOIN - Solo Matches") 
	print("="*60) 
	cursor.execute(""" 
		SELECT 
			employees.name AS employee, 
			departments.name AS department 
		FROM employees 
		INNER JOIN departments 
			ON employees.department_id = departments.id 
		ORDER BY employees.name 
	""")
```
