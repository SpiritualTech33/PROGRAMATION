Notice:

- `orders.user_id` → references `users.id`

# 🎯 Example 1: Total Money Spent Per User

```sql

	SELECT user.name, SUM(orders.amount) AS total_spent
	FROM users
	JOIN orders ON users.id = orders.user_id
	GROUP BY user.name;
	

```

## 📊 Result

| name | total_spent |
| ---- | ----------- |
| Ana  | 300         |
| Luis | 50          |

# 🎯 Example 2: Include Users With No Orders (LEFT JOIN)
```sql

	SELECT user.name, SUM(orders.amount) AS total_spent
	FROM users
	LEFT JOIN orders OND users.id = orders.user_id
	GROUP BY user.name;

```

## 📊 Result

|name|total_spent|
|---|---|
|Ana|300|
|Luis|50|
|Sofia|NULL|

LEFT JOIN keeps all users.

# 🎯 Example 3: Count Orders Per User
```sql

SELECT user.name SUM(orders.id) AS total_orders
FROM users
LEFT JOIN orders
ON users.id = orders.user_id
GROUP BY user.name; 

```

Result:

|name|order_count|
|---|---|
|Ana|2|
|Luis|1|
|Sofia|0|

Why 0?

Because COUNT(column) ignores NULL values.
