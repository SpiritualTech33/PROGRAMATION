# The Syntax Of Concurrency
---
We just saw what concurrency is, so now, we move into the pragmatic field, and we put it into practice with the Python's reserved words Async & Await.

`async` — The Declaration of Intent
When you write the word async right before a function, you're telling to Python: This is not a normal function, is a coroutine function. A function that knows how to pause itself and surrender the control to the loop event meanwhile waits.

```python
# Regular function — runs to completion, blocks everything
def fetch_data_from_server(server_url):
    response = make_network_request(server_url)  # Everything freezes here
    return response


# Async function — knows how to wait without blocking
async def fetch_data_from_server_async(server_url):
    response = await make_network_request(server_url)  # Pauses HERE, surrenders control
    return response
```

This is a coroutine function. A function with time awareness that knows when to pause and wait. 

When Python hits `await`, three things happen simultaneously:

1. **This coroutine pauses** — it stops executing
2. **Control returns to the event loop** — "I'm waiting, someone else can run"
3. **When the awaited thing finishes** — the event loop resumes this coroutine exactly where it paused


```python
import asyncio

async def make_three_requests_currently():
	"""
	Fetches three resources currently using async/await.
	
	Mental Model: We're launching 3 waiting processes simultaneously,
	then collecting results as they arrive.
	Not sequential. Not parallel. Concurrent
	"""
	
	#These coroutines are scheduled simultaneously
	first_result, second_result, third_result = await asyncio.gather(
		fetch_user_data_from_api(user_id=1),
		fetch_user_data_from_api(user_id=2),
		fetch_user_data_from_api(user_id=3)
	)
	
	return first_result, second_result, third_result
```

# 🔥 When Is Async Useful?

Async shines when tasks are:

- I/O bound
- Network calls
- Database queries
- File reads
- Web servers handling many users

It does NOT help with:

- Heavy CPU computation
- Matrix multiplication
- Training neural networks

For CPU-bound tasks → multiprocessing or C extensions.
