
# write() and writelines()
---
These are two ways to put content into a file. 

write() allow us to write a string, it doesnt make by himself the space, you must added it by yourself using \n
**Key point:** You control newlines. If you want a new line, use `\n`.
```python
#Wrong
with open("notes.txt", "w") as f:
    f.write("First line")
    f.write("Second line")  # These will be on SAME line!
# File contains: "First lineSecond line"



#Correct way
with open("notes.txt", "w") as f:
    f.write("First line\n")
    f.write("Second line\n")
# File contains two separate lines
```

writelines() take a list or any iterable of strings and write them sequentially. The new lines are not added by himself, so you must add them using \n
**Critical:** `writelines()` doesn't add newlines. Each string in your list must include `\n` if you want separate lines.

```python
#Wrong
lines = ["Line 1", "Line 2", "Line 3"]
with open("file.txt", "w") as f:
    f.writelines(lines)
# File contains: "Line 1Line 2Line 3" - all one line!

#Correct
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("file.txt", "w") as f:
    f.writelines(lines)
# File contains three separate lines
```

Practical Example

write()
```python
with open("training.txt", "w") as f:
    f.write("Monday: 100 swings\n")
    f.write("Tuesday: Sprint training\n")
    f.write("Wednesday: Rest\n")
```

writelines()
```python
# Building a training log
sessions = []
sessions.append("Monday: 100 swings\n")
sessions.append("Tuesday: Sprint training\n")
sessions.append("Wednesday: Rest\n")

with open("training.txt", "w") as f:
    f.writelines(sessions)  # Write all at once
```
