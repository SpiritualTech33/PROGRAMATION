# Why we can open text files on python? What's the point of it?
---

Well, this is a philosophical part of the subject by itself. We can open text files in Python, because that way we can give to our programs memory. When you're coding, all the things that you create in that program (variables, functions, classes) everything it's living in RAM memory. Once you close the program, all that info is deleted form his memory. So, the text files are a way to give our program a type of memory. Text files are human readable and that's why is so important to be able to manipulate text files. They are one of the most fundamental bridges between human and computational knowledge.
