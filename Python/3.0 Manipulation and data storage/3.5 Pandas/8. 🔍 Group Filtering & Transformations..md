## 🔍 Group Filtering

Group filtering means that you're now not only selecting single values, you're selecting an entire group and passing it through a filter. Remember the filtering from individual values that you practice before? well, now you can do the same to entire groups, not only isolated values. 

**Key Distinction:** 
- **Row filtering** → "Show me players with HR > 30" 
- **Group filtering** → "Show me teams whose average HR > 30"

**THE MECHANISM**
```python
# Syntax: 
grouped_data.filter(lambda group: condition_on_group) 
# The lambda receives an entire group (DataFrame) 
# It must return True (keep group) or False (discard group)
```

### EXAMPLES

```python
import pandas as pd
import numpy as np

# ===== CREATE A DATA FRAME =====

mlb_dataframe = pd.DataFrame({
	
	'player_name': [
		'Ohtani', 'Judge', 'Trout', 'Betts', 'Guerrero', 'Altuve',
		'Freeman', 'Machado', 'Tatis', 'Soto'
	],
	
	'team': [
		'Dodgers', 'Yankees', 'Angels', 'Dodgers', 'Blue Jays', 'Astros',
		'Dodgers', 'Padres', 'Padres', 'Mets'
	],
	
	'position': [
		'DH', 'RF', 'CF', 'SS', '1B', '2B', '1B', '3B', 'RF', 'RF'
	],
	
	'batting_avg': [
		0.298, 0.345, 0.301, 0.311, 0.302, 0.278, 0.286, 0.265, 0.287, 0.321 
	],
	
	'home_runs': [
		51, 62, 38, 29, 28, 22, 31, 27, 18, 42
	],
	
	'RBI': [
		111, 123, 98, 86, 101, 73, 85, 64, 59, 102
	],
	
	'games_played': [
		146, 138, 150, 139, 161, 98, 102, 125, 135, 158
	],
})


# FILTER: KEEP TEAMS WITH BA > .290
# Mental Model: Show me all players from elite hitting team
#=================================================================================
high_ba_teams = mlb_dataframe.groupby('teams').filter(
	lambda team_group: team_group['batting_avg'].mean() > 0.290
)



# FILTER 2: KEEP TEAMS WITH HOME RUNS > 40
# Mental Model: Show me the players from high power teams
#=================================================================================
high_hr_teams = mlb_dataframe.groupby('teams').filter(
	lambda team_group: team_group['home_runs'].sum() > 40
)



# FILTER 3: KEEP THE TEAMS WITH MORE THAN 1 PLAYER
# Mental Model: Exclude the teams with only one player
#=================================================================================
multuple_player_teams = mlb_dataframe.groupby('teams').filter(
	lambda team_group: len(team_group) > 1
)




# FILTER 4: COMPLEX CONDITION, HIGH AVG AND MORE THAN 40 HRs
# Mental Model: Teaams that hit for power and avg.
#=================================================================================
elite_hitting_teams = mlb_dataframe.groupby('teams').filter(
	lambda team_group:
	(team_group['hitting_avg'] > 0.290) &
	(team_group['home_runs'] > 40)
)

```

Now, you're creating filter for entire groups. 






## 🔄 Group Transformations

Is the act of transforming the entire group. Where not making smaller the data, we are enriching it

## 🎯 **KEY INSIGHT: When to Use What**

|Goal|Use|Output|
|---|---|---|
|"What's the average per group?"|**Aggregation**|One row per group|
|"How does each individual compare to their group?"|**Transformation**|Same rows, enriched|
|"Show me only high-performing groups"|**Filter**|Subset of original rows|
