
## 📊 `groupby()` - Aggregation and Analysis

Pandas groupby function is very similar to SQL function with the same name. Allows you to split group into categories, and then, apply  aggregation functions to each group. 

**SYNTAX**
```python
grouped_data = dataframe.groupby('column_to_group_by').aggregation_function()

# Common aggregation functions: 
# .mean() - average 
# .sum() - total 
# .count() - count of items 
# .min() / .max() - minimum/maximum 
# .std() - standard deviation
```

**EXAMPLES**
```python
mlb_datadrame = pd.DataFrame[{
	'player_name': ['Judge', 'Ohtani', 'Harper', 'Betts'],
	'team': ['Yankees', 'Dodgers', 'Phillies', 'Dodgers'],
	'AVG': [0.349, 0.298, 0.277, 0.301],
	'home_runs': [56, 51, 27, 29],
	'RBI': [102, 111, 87, 93]
}]


# ==== GROUP BY TEAM AVG =====

team_avg = mlb_dataframe.groupby('team')['AVG'].mean()


# ===== GROUP BY TOTAL HR FOR TEAM =====

team_hr = mlb_dataframe.groupby('team')['home_runs'].sum()


```


|Operation|Syntax|Use Case|
|---|---|---|
|Select by label|`.loc[row, col]`|"Give me 'price' column"|
|Select by position|`.iloc[row, col]`|"Give me 3rd column"|
|Slice rows (inclusive)|`.loc[1:5]`|"Rows 1 through 5"|
|Slice rows (exclusive)|`.iloc[1:5]`|"Rows 1, 2, 3, 4 (not 5)"|
|Single condition|`df[df['col'] > 5]`|"Where col > 5"|
|AND conditions|`df[(df['a'] > 5) & (df['b'] < 10)]`|"Where a>5 AND b<10"|
|OR conditions|`df[(df['a'] > 5) \| (df['b'] < 10)]`|"Where a>5 OR b<10"|
|NOT condition|`df[~(df['col'] == 5)]`|"Where col is NOT 5"|
|Group and aggregate|`df.groupby('col').mean()`|"Average by group"|
