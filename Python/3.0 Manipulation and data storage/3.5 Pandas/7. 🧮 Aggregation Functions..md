## 📊  Aggregation Functions - The Basic Toolkit

The aggregation functions are those functions that allows us to make operations with our data. Just like sum the total, count total values or receiving the max or the minimum value.

The most common are:

mean() --> Returns the average.
`mean = n∑xi​​`

sum() --> Returns the total sum of the values.
`sum = ∑xi​`

min()  --> Return the smallest value 

max() --> Return the highest value

count() --> Counts valid values (Not count Null values)

std() --> The standard deviation, it responds how dispersed are the data respect to the mean or average.
`std = n−11​∑(xi​−xˉ)2​`

median() --> The central value of the data.

```python
import pandas as pd

less_productive_players = mlb_dataframe.groupby('team')['RBI'].min()
```

## 🎨 Multiple Aggregations with `agg()` - The Power Move

Okey, we just saw how to make functions to our data. The main problem is that we are making one single operation per line. Imagine that you want the average per team, total home runs, total RBI and the highest average per team. You wouldn't do it 1 by 1, In this cases, you use agg(), a 
function that allows you to make multiple operations at once

**SYNTAX**

```python
# METHOD 1: List of functions (same function to all numeric columns)
df.groupby('column').agg(['mean', 'max', 'min'])

# METHOD 2: Dictionary (different functions to different columns)
df.groupby('column').agg({
    'col1': 'mean',
    'col2': ['sum', 'count'],
    'col3': ['min', 'max', 'std']
})

# METHOD 3: Custom functions
df.groupby('column').agg(lambda x: x.max() - x.min())
````





### EXAMPLE


```python

import pandas as pd
import numpy as np

# ===== CREATE A DATA FRAME =====

mlb_dataframe = pd.DataFrame({
	
	'player_name': [
		'Ohtani', 'Judge', 'Trout', 'Betts', 'Guerrero', 'Altuve',
		'Freeman', 'Machado', 'Tatis', 'Soto'
	],
	
	'team': [
		'Dodgers', 'Yankees', 'Angels', 'Dodgers', 'Blue Jays', 'Astros',
		'Dodgers', 'Padres', 'Padres', 'Mets'
	],
	
	'position': [
		'DH', 'RF', 'CF', 'SS', '1B', '2B', '1B', '3B', 'RF', 'RF'
	],
	
	'batting_avg': [
		0.298, 0.345, 0.301, 0.311, 0.302, 0.278, 0.286, 0.265, 0.287, 0.321 
	],
	
	'home_runs': [
		51, 62, 38, 29, 28, 22, 31, 27, 18, 42
	],
	
	'RBI': [
		111, 123, 98, 86, 101, 73, 85, 64, 59, 102
	],
	
	'games_played': [
		146, 138, 150, 139, 161, 98, 102, 125, 135, 158
	],
})



# METHOD 1: APPLY THE SAME FUNCTION TO ALL COLUMNS 
# Mental Model: Give me mean, max and min from all numeric stats per team
#=================================================================================
comprehensive_stats_per_team = mlb_dataframe.groupby('team').agg(
	['mean', 'max', 'min']
)



# METHOD 2: DIFFERENT FUNCTIONS TO DIFFERENT COLUMNS
# Mental Model: Customized analysis per metric
#=================================================================================
customized_analysis = mlb_dataframe.groupby('team').agg({
	'player_name': 'count', # How many players
	'batting_avg': ['mean', 'max'],  #The avg of the team and his best avg
	'home_runs': ['sum', 'mean', 'max'], # The Hr total, HR avg and max Hr
	'RBI': 'sum'   # The sum of the RBI's from a team
})


# METHOD 3: NAMED AGREGGATIONS
# Mental Model: Give me meaningful names as result
#=================================================================================
named_stats = mlb_dataframe.groupby('team').agg(
	total_players= ('player_name', 'count'),
	team_ba= ('batting_avg', 'mean'),
	total_hr= ('home_runs' 'sum'),
	best_slugger= ('home_runs', 'max'),
	team_rbis= ('RBI', 'sum')
	
)
```
