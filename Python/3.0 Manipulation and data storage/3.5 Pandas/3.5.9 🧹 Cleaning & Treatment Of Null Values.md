This is one of the mots common problems in real life data sets. The missing values.
Are the representation of absence of data. Is not a 0, is not a negative number, is simply something unknown, not data.

**Types of "nothingness" in Pandas:**

- `NaN` (Not a Number) - from NumPy, used for missing numeric data
- `None` - Python's null object
- `pd.NA` - Pandas' nullable type (newer)
- `pd.NaT` (Not a Time) - missing datetime values

### 👁️ How to IDENTIFY Null Values

```python
import pandas as pd
import numpy as np

# ===== CREATE A DATA FRAME WITH MISSING VALUES =====

players_data = pd.DataFrame({ 
	'player_name': ['Ohtani', 'Judge', 'Trout', 'Betts', 'Guerrero'], 
	'team': ['Angels', 'Yankees', None, 'Dodgers', 'Blue Jays'], 
	'batting_average': [0.304, 0.287, np.nan, 0.295, 0.311], 
	'home_runs': [46, 62, 40, None, 32], 
	'salary_millions': [30, 40, 35.5, np.nan, 28] 
})


# METHOD 1: .isnull() --> Returns boolean data frame
# Mental Model: "Show me True where data is missing, False where present"
#=================================================================================
missing_data_filter = players_data.isnull()


# METHOD 2: .isnull().sum() --> Count missing values per column
# Mental Model: "How many missing values in each column?"
#=================================================================================
missing_data_counter_per_column = players_data.isnull().sum()


# METHOD 3: .isnull().sum().sum()
# Mental Model: "How many total missing values in entire DataFrame?"
#=================================================================================
total_missing_data = players_data.isnull().sum().sum()


# METHOD 4: Missing percentage per column
# Menatl Model: "What % of each column is missing?"
#=================================================================================
total_rows = len(players_data)
missing_percentage = (players_data.isnull().sum() / total_rows) * 100


# METHOD 5: Identify rows with any missing value
# Mental Model: "Which rows have at least one missing value?"
#================================================================================
rows_with_any_missing = players_data[players_data.isnull().any(axis=1)]


# METHOD 6: Identify rows that are completly empty
# MEntal Model: "Which rows are completely empty?"
#=================================================================================
rows_with_all_missing = players_data[players_data.isnull().all(axis=1)]


```


Now, that we recognize the missing data, we can make strategies to solve that problem.


### 🛠️ STRATEGY 1: DELETE (Drop Missing Values)

 **When to Use:**
- Missing data is small percentage (< 5%)
- Missing data is random (not systematic)
- You have enough data to afford losing rows


## METHODS

```python

# ========================================
# DELETE STRATEGY 1: Drop rows with ANY missing value
# ========================================

# Mental model: "Remove any row that has at least one missing value"

cleaned_drop_any = players_with_missing_data.dropna()

print("After dropping rows with ANY missing value:")
print(cleaned_drop_any)
print(f"Rows remaining: {len(cleaned_drop_any)} out of {len(players_with_missing_data)}")
print("\n")


# ========================================
# DELETE STRATEGY 2: Drop rows with ALL missing values
# ========================================

# Mental model: "Remove only completely empty rows"

cleaned_drop_all = players_with_missing_data.dropna(how='all')

print("After dropping rows with ALL missing values:")
print(cleaned_drop_all)
print("\n")


# ========================================
# DELETE STRATEGY 3: Drop rows with missing in SPECIFIC columns
# ========================================

# Mental model: "I NEED these columns - drop if they're missing"

critical_columns = ['player_name', 'team']
cleaned_drop_subset = players_with_missing_data.dropna(subset=critical_columns)

print("After dropping rows with missing in critical columns:")
print(cleaned_drop_subset)
print("\n")


# ========================================
# DELETE STRATEGY 4: Drop COLUMNS with too much missing data
# ========================================

# Mental model: "If a column is >50% missing, remove the entire column"

threshold = 0.5  # 50%
missing_fraction = players_with_missing_data.isnull().sum() / len(players_with_missing_data)

columns_to_keep = missing_fraction[missing_fraction <= threshold].index
cleaned_drop_columns = players_with_missing_data[columns_to_keep]

print(f"Columns with >{threshold*100}% missing data removed:")
print(cleaned_drop_columns)
print("\n")


```




## 🔄 STRATEGY 2: REFILL (Impute Missing Values)

```python
# ========================================
# CREATE NEW SAMPLE WITH MORE DATA
# ========================================

players_for_imputation = pd.DataFrame({
    'player_name': ['Ohtani', 'Judge', 'Trout', 'Betts', 'Guerrero', 'Altuve'],
    'position': ['DH', 'RF', 'CF', 'RF', '1B', '2B'],
    'batting_average': [0.304, 0.287, np.nan, 0.295, 0.311, 0.300],
    'home_runs': [46, 62, 40, np.nan, 32, 28],
    'salary_millions': [30, 40, np.nan, 45, 28, 32]
})

print("Original data with missing values:")
print(players_for_imputation)
print("\n")


# ========================================
# REFILL METHOD 1: Fill with a constant value
# ========================================

# Mental model: "Replace missing with a specific value"

filled_with_zero = players_for_imputation.copy()
filled_with_zero['home_runs'] = filled_with_zero['home_runs'].fillna(0)

print("Filled home_runs with 0:")
print(filled_with_zero)
print("\n")


# ========================================
# REFILL METHOD 2: Fill with MEAN (average)
# ========================================

# Mental model: "Replace missing with column average"
# BEST FOR: Numeric data with normal distribution

filled_with_mean = players_for_imputation.copy()
batting_avg_mean = filled_with_mean['batting_average'].mean()
filled_with_mean['batting_average'] = filled_with_mean['batting_average'].fillna(batting_avg_mean)

print(f"Filled batting_average with mean ({batting_avg_mean:.3f}):")
print(filled_with_mean)
print("\n")


# ========================================
# REFILL METHOD 3: Fill with MEDIAN (middle value)
# ========================================

# Mental model: "Replace missing with middle value"
# BEST FOR: Numeric data with outliers (more robust than mean)

filled_with_median = players_for_imputation.copy()
salary_median = filled_with_median['salary_millions'].median()
filled_with_median['salary_millions'] = filled_with_median['salary_millions'].fillna(salary_median)

print(f"Filled salary with median ({salary_median:.1f}):")
print(filled_with_median)
print("\n")


# ========================================
# REFILL METHOD 4: Fill with MODE (most common)
# ========================================

# Mental model: "Replace missing with most frequent value"
# BEST FOR: Categorical data

filled_with_mode = players_for_imputation.copy()
most_common_position = filled_with_mode['position'].mode()[0]
# Note: We won't fill position here as it has no missing values, just showing syntax

print(f"Most common position: {most_common_position}")
print("\n")


# ========================================
# REFILL METHOD 5: Forward fill (propagate last valid)
# ========================================

# Mental model: "Use the last known value"
# BEST FOR: Time series data

filled_forward = players_for_imputation.copy()
filled_forward['batting_average'] = filled_forward['batting_average'].fillna(method='ffill')

print("Forward filled batting_average:")
print(filled_forward)
print("\n")


# ========================================
# REFILL METHOD 6: Backward fill (propagate next valid)
# ========================================

# Mental model: "Use the next known value"

filled_backward = players_for_imputation.copy()
filled_backward['batting_average'] = filled_backward['batting_average'].fillna(method='bfill')

print("Backward filled batting_average:")
print(filled_backward)
print("\n")


# ========================================
# REFILL METHOD 7: Fill with GROUP mean (smart imputation)
# ========================================

# Mental model: "Replace missing with average of similar players"
# BEST FOR: When groups have different characteristics

# Fill home_runs missing values with position average
filled_with_group_mean = players_for_imputation.copy()
filled_with_group_mean['home_runs'] = (
    filled_with_group_mean.groupby('position')['home_runs'].transform(
        lambda x: x.fillna(x.mean())
    )
)

print("Filled home_runs with position average:")
print(filled_with_group_mean)
print("\n")


# ========================================
# REFILL METHOD 8: Interpolate (estimate based on neighbors)
# ========================================

# Mental model: "Estimate missing value from surrounding values"
# BEST FOR: Sequential/ordered data

filled_interpolated = players_for_imputation.copy()
filled_interpolated['salary_millions'] = filled_interpolated['salary_millions'].interpolate()

print("Interpolated salary values:")
print(filled_interpolated)
print("\n")
```



## 🏷️ **STRATEGY 3: MARK (Flag Missing Values)**

 **When to Use:**
- Missingness itself is informative
- You want to keep original data but track what was missing
- You'll use advanced ML that handles missing data


```python
# ========================================
# MARK METHOD 1: Create boolean indicator column
# ========================================

# Mental model: "Add a flag showing where data was missing"

marked_data = players_for_imputation.copy()

# Create indicator for each column with missing values
marked_data['batting_average_was_missing'] = marked_data['batting_average'].isnull()
marked_data['home_runs_was_missing'] = marked_data['home_runs'].isnull()
marked_data['salary_was_missing'] = marked_data['salary_millions'].isnull()

# Now fill the missing values (so you have both the filled value AND the flag)
marked_data['batting_average'] = marked_data['batting_average'].fillna(
    marked_data['batting_average'].mean()
)
marked_data['home_runs'] = marked_data['home_runs'].fillna(
    marked_data['home_runs'].mean()
)
marked_data['salary_millions'] = marked_data['salary_millions'].fillna(
    marked_data['salary_millions'].median()
)

print("Data with missing value indicators:")
print(marked_data)
print("\n")


# ========================================
# MARK METHOD 2: Use special sentinel value
# ========================================

# Mental model: "Replace missing with special value like -999"
# (Less common, but useful in some ML contexts)

marked_with_sentinel = players_for_imputation.copy()
marked_with_sentinel['home_runs'] = marked_with_sentinel['home_runs'].fillna(-999)

print("Marked missing home_runs with -999:")
print(marked_with_sentinel)
print("\n")
```
