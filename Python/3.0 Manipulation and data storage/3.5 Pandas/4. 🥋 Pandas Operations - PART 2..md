## 🔪  Slicing - Range Selection

Slicing is an operation that give us the chance to select continuos snippets of rows or columns.
It's like when you cut bread. You don't cut separated parts, you cut an entire slice to eat. Here is the same, yo can cut slices of your data frame. 

To slice, you can use both tools, `.loc[]` & `.iloc[]` but there are slight differences between them that you must understand. 

**BASIC SYNTAX**
```python
dataframe.loc[start_label:end_label]
# There's a key point here.
# The en label is inclusive.
# Which means that the label at the end becomes part of the slice.
# We could say that .loc[] is inclusive.

dataframe.iloc[start_pos:end_pos]
# Here, the end position is exlusive.
# Wich means that the end position it doesn't become part of the slice.
# We could say then, that .iloc[] is exclusive.
```

**EXAMPLES - SLICING IN ACTION**

`.loc[]`
```python
mlb_datadrame = pd.DataFrame[{
	'player_name': ['Judge', 'Ohtani', 'Harper', 'Betts'],
	'team': ['Yankees', 'Dodgers', 'Phillies', 'Dodgers'],
	'AVG': [0.349, 0.298, 0.277, 0.301],
	'home_runs': [56, 51, 27, 29],
	'RBI': [102, 111, 87, 93]
}]


# ===== SLICING ROWS =====

players_slice_loc = mlb_dataframe.loc[0:3]

# I choose rows from 0 to 3
# Remember that loc is inclusive, so te row 4 is gonna be in the slice


# ===== SLICING COLUMNS =====

players_column_slice = mlb_dataframe.loc[:, 'team':'home_runs']

# Here i choose from the team column to the home runs one.
# I didn't pick rows, so i will have all then rows.
```


`iloc[]`
```python
mlb_datadrame = pd.DataFrame[{
	'player_name': ['Judge', 'Ohtani', 'Harper', 'Betts'],
	'team': ['Yankees', 'Dodgers', 'Phillies', 'Dodgers'],
	'AVG': [0.349, 0.298, 0.277, 0.301],
	'home_runs': [56, 51, 27, 29],
	'RBI': [102, 111, 87, 93]
}]

# ===== SLICING ROWS =====

selected_rows = mlb_dataframe.iloc[0:3]

# Here i choose the same range from ,loc[] example
# But .iloc[] is exclusive
# So my actual range would be from row 0 to 2


# ===== SLICING COLUMNS =====

selected_columns = mlb_dataframe.iloc[:, 0:3]

# Here i choose the columns from 0 to 3
# But how .iloc[] is esclusive, it will take from columns 0 to 2
# So, i take the columns name, team, and AVG, exluding home runs.
```

## 🎯 Conditional Filtering - The Power Move

Okey, we just saw how to select specific columns and rows, by choosing. But now, we gonna learn how to filter the info. Select just the info that has certain criteria. 

**MECHANISM - HOW BOOLEAN INDEXING WORKS**
```python
# Step 1: Create a boolean mask (True/False for each row)
mask = dataframe['column'] > value

# Step 2: Use that mask to filter
filtered_data = dataframe[mask]

# Or combine in one line:
filtered_data = dataframe[dataframe['column'] > value]
```

**EXAMPLES: Single Condition Filtering**
```python
mlb_datadrame = pd.DataFrame[{
	'player_name': ['Judge', 'Ohtani', 'Harper', 'Betts'],
	'team': ['Yankees', 'Dodgers', 'Phillies', 'Dodgers'],
	'AVG': [0.349, 0.298, 0.277, 0.301],
	'home_runs': [56, 51, 27, 29],
	'RBI': [102, 111, 87, 93]
}]

# ====== Filter batters with AVG > .300 =====

# STEP 1: Create the boolean mask or filter
avg_filter = mlb_dataframe['AVG'] > 0.300

# STEP 2: Apply the filter
elite_hiiters = mlb_dataframe[avg_filter]


# ===== FILTER HITTERS WITH MORE THAN 40 HR =====

# STEP 1: CREATE THE FILTER
hr_filter = mlb_dataframe['home_runs] > 40

# STEP 2: APLLY THE FILTER
sluggers = mlb_dataframe[hr_filter]

# OR DIRECTLY APPLY THE FILTER
power_hitters = mlb_dataframe[
mlb_dataframe['home_runs'] >= 40 
]


```



## 🔗  Multiple Conditions - Boolean Logic

We just saw how to filter data with a single condition. But we can also create filters with multiple conditions to make more exclusive filters. 


 **THE OPERATORS:**
- **AND** → `&` (both conditions must be True)
- **OR** → `|` (at least one condition must be True)
- **NOT** → `~` (negation - flip True/False)


**CRITICAL: You MUST use parentheses around each condition.**
```python
# ❌ WRONG (will cause error):
df[df['col1'] > 5 & df['col2'] < 10]

# ✅ CORRECT:
df[(df['col1'] > 5) & (df['col2'] < 10)]
```

**EXAMPLES WITH MULTIPLE CONDITIONS**
```python

mlb_datadrame = pd.DataFrame[{
	'player_name': ['Judge', 'Ohtani', 'Harper', 'Betts'],
	'team': ['Yankees', 'Dodgers', 'Phillies', 'Dodgers'],
	'AVG': [0.349, 0.298, 0.277, 0.301],
	'home_runs': [56, 51, 27, 29],
	'RBI': [102, 111, 87, 93]

# FILTER HITTERS WITH HIGH AVG AND HIGH HR

complete_hitters = mlb_dataframe[
(mlb_dataframe['AVG'] > 0.300) & (mlb_dataframe['home_runs'] > 40)
]

# FILTER PLAYER WITH YANKEES OR PHILLIES

ny_or_philadeplphia_players = mlb_dataframe[
(mlb_dataframe['team'] == 'Yankees') | (mlb_dataframe['team'] == 'Phillies')
]

# FILTER PLAYERS THAT DON'T BELONG TO  DODGERS

dodgers_players = mlb_dataframe[
~(mlb_dataframe['team'] == 'Dodgers')
]
```
